# my-public-api

Prosty projekt Express w TypeScript.

## Struktura

- `src/app.ts` — konfiguracja aplikacji Express
- `src/server.ts` — uruchomienie serwera
- `tsconfig.json` — konfiguracja TypeScript
- `package.json` — skrypty i zależności

## Skrypty

- `npm run dev` — uruchomienie w trybie deweloperskim
- `npm run build` — kompilacja do `dist/`
- `npm start` — uruchomienie zbudowanej aplikacji
